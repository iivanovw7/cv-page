`First` is to find out the process ID.
```bash
    sudo lsof -n -i:<port number>;
```
<br />

Same for Mac:
```bash
    lsof -i tcp:<port no>;
```
<br />

`Second` is to kill application by ID.
```bash
    kill -9 <PID>;
```
<br />
