Check DNS leak <Link href="https://dnsleaktest.com" text="Here" />
If there are any problems (<Link href="https://github.com/dnscrypt" text="solution" />):
<br />

Purge:
```bash
    sudo apt purge dnscrypt-proxy
```
<br />

Install:
```bash
    sudo apt update
    sudo apt install dnscrypt-proxy
```
<br />

Now you'll need to change your DNS server to 127.0.2.1. <br />
Restart  network services:
```bash
    sudo systemctl restart NetworkManager
```
